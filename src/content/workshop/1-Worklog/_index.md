## Worklog

Báo cáo cá nhân theo tuần trong 12 tuần thực tập tại FCAJ trên dự án **Upscale AI**. Vai trò của tôi: **Team Lead** — lên kế hoạch, chia việc, điều phối nhóm, review, và tự tay làm một phần (kiến trúc tổng thể, spec, ADR, tài liệu).

### Cấu trúc

Sáu tuần đầu chủ yếu là học và làm lab theo chương trình FCAJ. Từ tuần 7 mới bắt tay vào dự án Upscale AI.

| Tuần | Trọng tâm | Nội dung học / làm | Linear |
| --- | --- | --- | --- |
| 1 | Onboarding FCAJ, tài khoản AWS | Console, Regions/AZ, Budgets, Support, lab đầu tay | UPS-1 |
| 2 | IAM + VPC/EC2 cơ bản | User/Group/Role/Policy, MFA, Least Privilege, dựng VPC + EC2 | UPS-2 |
| 3 | IAM Role cho EC2, S3 static website | Cloud9, Bucket Policy, Static Hosting | UPS-3 |
| 4 | RDS + Lightsail | Tạo DB, kết nối app, managed services | UPS-4 |
| 5 | CloudWatch + AWS CLI | Log, metric, alarm, quản lý bằng CLI | UPS-5 |
| 6 | DynamoDB + ElastiCache | NoSQL, Redis cache | UPS-6 |
| 7 | Khởi động Upscale AI | Kiến trúc, scaffold React+Vite+TanStack + FastAPI | UPS-7 |
| 8 | Upload ảnh | UI upload, API upload, validate, luồng FE↔BE | UPS-8 |
| 9 | Real-ESRGAN + CodeFormer | Tích hợp mô hình upscale + phục hồi khuôn mặt | UPS-9 |
| 10 | Docker + ECS/Fargate + S3 | Image FE/BE, deploy lên AWS, lưu ảnh S3 | UPS-10 |
| 11 | SQS + Redis + SSE | Hàng đợi bất đồng bộ, cache, tiến trình realtime | UPS-11 |
| 12 | Hoàn thiện, demo, cleanup | Test, tài liệu, báo cáo, dọn tài khoản | UPS-12 |
